# Hero Scroll Minimal (Text-only, Black)

A tiny, ready-to-run Vite + React + TypeScript + Tailwind + Framer Motion starter with a **text-only scroll-motion hero** and minimal UI.

## Quickstart

```bash
# 1) Install deps
npm install

# 2) Run dev server
npm run dev

# 3) Build
npm run build

# 4) Preview production build
npm run preview
```

## Testing (optional)
This template includes a couple of inline Vitest tests inside `src/HeroScroll.tsx`.
They **only run in Vitest** (they are ignored in regular builds).

```bash
npm run test
```

## Notes
- The hero uses scroll-driven motion via Framer Motion (`useScroll`, `useTransform`, `useMotionTemplate`).
- `prefers-reduced-motion` is respected.
- Pure black background, text-only hero, minimal layout.
